# 执行标准化部署流程

在这一步，我们将使用WeCube的任务编排功能执行标准化的部署流程。


## 使用任务编排固化标准化操作流程

请通过 “**协同**菜单” - “**任务编排**菜单项” 进入任务编排管理页面。


## 执行用于应用部署的任务编排

请通过 “**执行**菜单” - “**编排执行**菜单项” 进入任务编排执行页面。
